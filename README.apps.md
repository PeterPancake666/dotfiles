# Apps

## Necesary apps

### - Qtile (WM)
```bash
pacman -S qtile
```

### - Alacritty (Terminal)
```bash
pacman -S alacritty
```

### - Tmux (Terminal Multiplexer)
```bash
pacman -S tmux
```

### - Neovim (Text Editor)
```bash
pacman -S neovim
```

## zsh

### - zsh (shell)
```bash
pacman -S zsh
chsh -s zsh
```
### - ohmyzsh (zsh config)
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
### - powerlevel9k (ohmyzsh theme)
```bash
paru -S zsh-theme-powerlevel9k-git
```
